# 在 LobeChat 中使用 302.AI

<Image cover src={'https://github.com/user-attachments/assets/854471db-1fa5-4650-b3b3-b2a7377e5bb3'} />

[302.AI](https://www.302.ai/) 是一个按需付费的 AI 应用平台，提供市面上最全的 AI API 和 AI 在线应用。

本文将指导你如何在 LobeChat 中使用 302.AI。

<Steps>
  ### 步骤一：获得 [302.AI](https://www.302.ai/) 的 API Key

  - 点击 `开始使用`，注册并登录 [302.AI](https://www.302.ai/)
  - 点击左侧的 `API Keys`
  - 点击 `添加API KEY`，复制并保存生成的 API 密钥

  <Image alt={'获取 API 密钥'} inStep src={'https://github.com/user-attachments/assets/854471db-1fa5-4650-b3b3-b2a7377e5bb3'} />

  ### 步骤二：在 LobeChat 中配置 302.AI

  - 访问 LobeChat 的`设置`界面
  - 在`语言模型`下找到 `302.AI` 的设置项

  <Image alt={'填入 API 密钥'} inStep src={'https://github.com/user-attachments/assets/767e3b95-a9c1-43d5-81b9-4b36519a788e'} />

  - 填入获得的 API 密钥
  - 为你的 AI 助手选择一个 302.AI 的模型即可开始对话

  <Image alt={'选择 302.AI 模型并开始对话'} inStep src={'https://github.com/user-attachments/assets/4c933151-42e8-46db-868a-3eb394995aed'} />

  <Callout type={'warning'}>
    在使用过程中你可能需要向 API 服务提供商付费，请参考 302.AI 的相关费用政策。
  </Callout>
</Steps>

至此你已经可以在 LobeChat 中使用 302.AI 提供的模型进行对话了。
